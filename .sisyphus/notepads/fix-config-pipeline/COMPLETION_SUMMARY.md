### Architecture Restored

```
User's provider module (exports provideConfig function)
         ↓
.generated.tailwind.config.js  ← physical file on disk (generated by plugin)
  - imports provideConfig from user's module
  - inlines originalConfig (nested Tailwind structure)
  - calls provideConfig(originalConfig)
  - _flattenColors() flattens nested → flat keys
  - exports __twConfig (flat keys for runtime refs)
         ↓
Component modules import __twConfig from .generated.tailwind.config.js
```
